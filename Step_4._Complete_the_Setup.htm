Step 4. Complete the setup

After completing the setup, the Collector starts automatically and runs as a service. The Collector also starts automatically after rebooting.

To ensure the files are coming in to Sumo Logic, click the Status tab to check the current status for the installed Collector. After a few moments, the status of your Collector will be listed as Running. If you set up a Source in Step 3, the status will also show a number of collected messages.

If you have waited 15 to 30 seconds from the time you configured the Collector and the number of messages is still listed as zero, check:

  • The Source configuration path settings to make sure your Collector has a valid access path to the local or remote files you want to collect. Make sure you entered an absolute path from the root, or for Windows, from the drive letter.
  • The SSH key path is correct if you are using SSH to authenticate to remote servers for remote file collection.
  • For Syslog collections, ensure the syslog sources are pointing to the same port that the Syslog Source is listening on.
  • For collections from Windows machines, see Prerequisites for Windows Collections.
