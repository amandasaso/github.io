Scheduling searches

There are two ways of setting up scheduled searches. You can elect to receive a summary email, where the subject of the email is titled Search Results. This type of email contains a summary of the search results for your information or use. The email will arrive as scheduled, depending on the frequency the search is run. Scheduled searches can be edited or canceled at any time.

Alternately, you can use a scheduled search as a method of setting an alert so you're notified only when certain conditions arise. These emails, with the subject of Search Alert, will only arise when the conditions you set are met. For example, you could set up an alert if a certain number of users visit the free trial URL of your website. You can learn more about using conditions to set up alerts in Using scheduled searches. You can run a search from this email as well.

To schedule a search:

  1. Do one of the following:
  • Open a saved search and convert it to a scheduled search. Click the Open link below the Search query field. Select the search you want to schedule, and then click Edit.
  • Use your current search query and then schedule it to periodically run and send alerts by email.
  1. Choose an option from the Run frequency menu:

Never. Choose this option to temporarily turn off scheduled search.

Every 15 minutes. The search will run for the first time when you save the schedule, and then every 15 minutes after that.

Hourly. The search will run for the first time at the top of the next hour after you save the schedule, and then every hour after that.

Daily. Choose the time you'd like to run the search every day. A Daily search will cover exactly 24 hours of activity. You can change the schedule whenever you'd like.

  • Select Notify me every time upon search completion if you want an email with search results every time the search is run (depending on the frequency, you could get an email every 15 minutes, every hour, or once a day).
  • If you'd like to set up a scheduled search that will alert you to specific events, choose Notify me only if the condition below is satisfied, and then set conditions before typing a value in the text box:

Number of Messages or Number of Groups. Depending on the search, set a condition to receive an email by the number of log messages or the number of user groups.

Equal to. Choose if there is an exact number of records in a search result at which you want to be notified.

Greater than. Choose if you want to be notified only if the search results include greater than that number of messages or groups you set in the text box.

Greater than or equal to. Choose if you'd like to be notified only if the search brings up greater than or equal to that number of messages or groups.

Fewer than. To be notified only if a search brings up less than the desired number of records, choose this option.

Fewer than or Equal to. Choose if you'd like to be notified only if the search brings up greater than or equal to that number of messages or groups.

  1. For Recipients, enter one or more email addresses separated by commas for the recipients of the search results. You can edit the recipients at any time.
  2. Click Save. An email notification will be sent to the recipient email address(es) based on the notification parameters. For queries returning high numbers of messages or group results, the email notification contains a representative sample of the first 20 messages or groups.
