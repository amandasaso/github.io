Ternary Expressions: Using 'If' or '?'

There are two forms of ternary expression you can use in Sumo Logic queries: one is constructed with the question mark (?) operator, and the other is constructed using the IF operator. The syntax varies slightly, but the results are equivalent. You can use the syntax you're most comfortable with.

Ternary expressions are used to evaluate a condition as either true or false, with values assigned for each outcome. It is a shorthand way to express an if-else condition. On the basis of the test, the entire expression returns value_if_true if condition is true, else value_if_false if the condition is false. The two sub-expressions (value_if_true and value_if_false) must have the same type.

To construct a ternary expression using the IF operator:

  • Syntax: ... | if(condition, value_if_true, value_if_false) as alias_field | ...
  • Examples:
    • ... | if(status_code matches "5*", 1, 0) as servererror | ...
    • ... | if(status_code matches "2*", 1, 0) as success | ...

To construct a ternary expression using the question mark (?) operator:

  • Syntax: ... | alias_field = condition ? value_if_true : value_if_false | ...
  • Examples:
    • disk_usage > threshold ? "disk full" : "OK" as status
    • a < b ? a : b      // This is the same as min(a, b)
