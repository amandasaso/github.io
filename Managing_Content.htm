Adding content

With the Content Library you can save, edit, delete, and publish (share) searches and Dashboards. Of course you can launch a search and open a Dashboard with a single click.

Adding a search

Searches are added to the Content Library from the Search page. Once you type a query in the search text box, you can add it to the library.

To add a search:

  1. In the Search tab, after typing your search query, click Save As below the search field.
  2. In the Save Search dialog, enter a name for the search. You can also make edits to the query if you’d like. Either add the search to the Saved folder or choose another folder.
  3. If you'd like to set up the search to run on a schedule, choose those options. Learn more here.
  4. Click Save.

If you’d like to create a new folder, click the blue “+” icon.

Adding a Dashboard

After you run a search, you can add it as a Monitor to a Dashboard in the Content Library, either to an existing Dashboard or in a new Dashboard.

To add a Dashboard:

  1. Run a search. Once the results are complete, click the Add to Dashboard icon.
  2. In the Add to Dashboard dialog, either choose an existing Dashboard or type a name of a new Dashboard.
  3. Choose a folder where you’d like to store the Dashboard. If you’d like to create a new folder, click the blue “+” icon.
  4. There are two column layouts: the standard column, where all columns are a uniform width, or the layout option (currently named Experimental) where each Monitor can have a different width. Choose Experimental, or to use the standard layout, choose an option from the Column menu.
  5. Type a title for the Monitor.
  6. Click Add.
