Adding a New Source

During the setup process, you probably created your first Source. You can add more Sources to your Collectors at any time. Each Source is tagged with its own metadata (Source Host, Source Category and Source Name). The more Sources you set up, the easier it is to isolate one of the Sources in a Search since each Source can be identified by its metadata tags.

To add a new Source:

  1. In the Sumo Logic Web Application, click the Collectors tab.
  2. Locate the Collector where you'd like to add a new Source and then click Add Source.
  3. Select the type of Source you'd like to add to this Collector: Local File, Remote File, Syslog, or Windows Events.
  4. Configure the Source. For instructions on configuring the various Source types, see:
    • Local File Source
    • Remote File Source
    • Syslog Source
    • Windows Events Source
    • Script Source
