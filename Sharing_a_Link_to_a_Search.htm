Sharing a Link to a Search

A search query can be shared ad-hoc by sending another user in your organization a Search Code. Clicking the Share Search Code icon creates a link that you can copy and paste. Then, the recipient can paste the link in the Search text box and run the search. Please note that search results may vary depending on each user's permissions.

To share a link to a search:

  1. Run a search query you'd like to share.
  2. After the search results are complete, click the Share Search Code icon under the results.



  3. Do one of the following:
  • Copy and paste the link to share via email or IM. This link can be pasted into a browser; a user will be prompted to log in to Sumo Logic if he or she is not currently logged in.
  • If you know that the recipient is logged in to Sumo Logic, copy and paste the bottom code. This code can only be pasted into the Search text box in the Sumo Logic Web Application.
