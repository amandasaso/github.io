Installation using the Collector Management API

The newest method that allows you to install Collectors and configure Sources across a deployment is our Collector Management API. The Collector Management API supports all of the configuration options found in the Source setup UI--including filters. In addition, you can flag a Collector as ephemeral or clobber the Collector via the API.

The API's JSON syntax means that you don't have to migrate any JSON parameters you're currently passing using the existing JSON files found in the installation directory.

To learn more, see the Collector Management API documentation.
