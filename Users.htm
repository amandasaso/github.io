Managing Users and Roles

In the Users tab of the Sumo Logic Web Application, you can view a list of users with access to your company data in the Sumo Logic Cloud. You can also provision new users, and delete or modify existing users.

What do you want to do?

  • Add a new user or edit details for an existing user
  • Temporarily deactivate a user
  • Delete a user
  • Reset a user's password
