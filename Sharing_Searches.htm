Sharing a Search in the Search Library

Any saved search can be shared with the other users in your organization. When you share a search, the search appears by name in the Shared Searches library for all the Sumo Logic users in your organization. Anyone can run the search, but only you as the owner can edit the shared search query.

To share a search:

  1. Do one of the following:
  • To share your current search, click Save As below the search field. Type a name for the search in the dialog box.



  • To share a saved search, click Open below the search query field to open your list of saved searches. Click Edit next to the search you want to share.


  1. Select the checkbox to Share search.


  2. Click Save. The search you shared now appears in the Shared Search library for your organization.

To stop sharing a search:

  1. Open the saved search and uncheck the Share search option.
  2. Click Save. The shared search is removed from the Shared Searches library for your organization.

To run a shared search:

  1. Click the book icon at the left side of the search query dialog.
  2. From the left-column, select Shared Searches, and then select the search you want to run.



  3. The shared search query appears in the query field. Hit enter or click Start to run the search.

To save a shared search to your own Saved Searches:

  1. Select the search from the Shared Searches list.
  2. Click Save As below the search query field.
  3. Give the search a new name. After it is saved, you can:
    • Edit the query.
    • Schedule the query to run and send you a search alert.
