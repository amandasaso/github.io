What's the difference between Collector types?

Installed Collectors and Hosted Collectors are used in much different ways:

 

  • Installed on a system within your deployment.
  • Can be configured with any of the following Source types: Local File, Remote File, Syslog, Local Windows Event Logs, Remote Windows Event Logs, and Script.
  • Hosted by Sumo Logic; no software to install or activate on a system in your deployment.
  • Can be configured with Amazon S3 Sources or HTTP Sources to upload data.
