Using the Content Library

The Content Library provides a central location for shared and saved content in your Sumo Logic account, as well as content shared by others in your organization. In addition to shared and saved searches, Dashboards can be saved and shared in the Content Library.

In addition to being a great tool for managing content, the Content Library allows you to launch searches and Dashboards with a single click--speeding up access to the searches you find yourself running consistently. Additionally, you can use the content that others in your organization have already developed to continually discover new insights in your data.
