Using Dashboards in Sumo Logic Free Accounts

 

Dashboards in Sumo Logic Free accounts work the same was as those in Enterprise accounts with one limitation: organizations with Sumo Logic Free accounts have access to 20 Monitors, shared between the three users in the account. You can decide to build as many Dashboards as you'd like with the available Monitors in your account.

Can I share Dashboards in a Sumo Logic Free account?

Yes. Sharing Dashboards works the same in Sumo Logic Free and Enterprise accounts. Once you share a Dashboard it's made available to all the users in the account.

Sumo Logic Free Default Dashboards

To help you get up and running with Dashboards, Monitors are preconfigured in a OS-optimized Default Dashboard that automatically populate with relevant ops and event insights gathered from your data. Default Dashboards can be modified and shared, just as customized Dashboards you build.

Only one Default Dashboard appears in a Sumo Logic Free account. Sumo Logic looks at the data received from the first Collector you installed to determine which Default Dashboard should be displayed.

Can I customize Monitors in the Default Dashboard?

Yes. You can change the queries saved as Monitors, or even delete a Monitor. Note that modifying a preconfigured Monitor doesn't free up an additional Monitor, but instead takes the place of the preconfigured Monitor.
