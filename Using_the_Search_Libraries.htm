Running queries from search libraries

To make it easier to access all of your search queries, you can choose a search from a search library. Search libraries include:

  • Recent Searches — A list of the last 32 searches you've run from the search query field.
  • My Saved Searches — Searches you save and name are listed in the Saved Searches list. See Saving a search for more information.
  • Shared Searches — These are searches shared by other Sumo Logic users in your organization. See Sharing a search for information about how to share a search with other people.

To run a search using any of the search queries listed in the search library:

  1. Click the saved search icon, then click Recent Searches, My Saved Searches, or Shared Searches to view each library.



  2. Click a query to run it.
