Resetting a User's Password

Users can reset their own passwords, or passwords can be reset by Administrators at any time.

To reset a user's password as an Administrator:

  1. In the Sumo Logic Web Application, click the Users tab.
  2. Find the user name for the user whose password you want to reset, and then click Edit.
  3. At the bottom of the Edit User dialog, click the link for Reset Password.



  4. Confirm the password reset by clicking OK.

A new random password is be generated and sent to the user at the email address listed in the user's details. After the user logs in with the reset password, he is immediately prompted to enter a password of his own choosing.
