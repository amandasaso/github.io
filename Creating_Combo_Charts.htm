Creating combo chart Monitors

Combo charts allow you to create bar or column chart with a secondary vertical axis so you can add a separate line as a comparison. For example, say we're running query like:

error | 5 as a| 7 as b | 7 as c | 7 as d | avg(a, b), max (c,d), min(c), count by _source

Generally, this query would produce a bar or column chat that looks a bit like this:

But with a combo chart, we can set the _count volume to display as a line chart on a second axis, so the count value is represented by a line instead of a bar:

Combo charts can be added to a Dashboard or can be viewed in the Aggregates pane.

To create a combo chart:

  1. Run a query
  2. In the Aggregates tab, choose bar chart or column chart to display the search results.


  3. To add a second axis, click the properties icon Monitor or Aggregate pane, then choose Change Series.



  4. In the Change Series dialog box, choose the value you'd like to add as a secondary axis and chart type. In our example, we set the _count value as a Line Chart on the Secondary vertical axis. Click Save.



    The combo chart is built and displayed.
