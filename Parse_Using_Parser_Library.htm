Using a Parser Library

Parser Libraries all you to apply a preconfigured library of parsing rules to evaluate messages according to pattern matches written as regular expressions.

Currently Sumo Logic offers Parser Libraries for the following:

  • Apache Access Logs
  • Apache Error Logs
  • Cisco ASA Logs
  • Microsoft IIS Logs

Additional parsers will soon be available! Check back often.
