Adding a User or Editing User Details

From the Users tab of the Sumo Logic Web Application, you can add new users or edit the detailed settings for an existing user.

Email addresses cannot be edited after a user is created. If a user's email address changes, the account administrator will need to create a new user account for that person.

To add a user:

  1. Click Manage > Users in the Sumo Logic Web Application.



  2. Do one of the following:
  • If the Manage Roles page is displayed, click View Users (to display the Manage Users page), then click New User.
  • If the Manage Users page is displayed, click New User.
  1. Click New User.
  2. Enter the user's details:
  • First and Last Name. Enter the first and last name of the user.
  • Email Address. The email address can only be entered for new users and must match the corporate domain of the Sumo Logic account. The email address cannot be edited later. For new users, Sumo Logic automatically sends a temporary password to the user at this email address after the user is created. The first time the user logs into Sumo Logic, he will be prompted immediately to change the temporary password to a password of his choice.
  • Status. Select Active or Inactive for the user status. You might choose Inactive for a new user who has not yet started work at the company, or to prevent access for an existing user. Click Active at any time to change the status back. Users are not notified of status changes. To completely remove a user, see Deleting a User.
  • Roles. Select a role or change the role for this user. Administrators can add and edit user settings. Analysts cannot add or edit users.
  1. Click Save.

To edit settings for an existing user:

  1. Click the Users tab in the Web Application.
  2. Click the Edit link to the right of the user's name.
  3. Edit the user's details:
  • First and Last Name. Enter or change the first and last name of the user.
  • Status. Select Active or Inactive for the user status. You might choose Inactive for a new user who has not yet started work at the company, or to prevent access for an existing user. Click Active at any time to change the status back. Users are not notified of status changes. To completely remove a user, see Deleting a User.
  • Roles. Select a role or change the role for this user. Administrators can add and edit user settings. Analysts cannot add or edit users.
