Step 3. Add a Source

After setting up an Installed Collector you'll add Sources. If you set up a Source when you installed your Collector, you can create an additional Source type.

It's easy to add more Sources later from the Collectors tab of the Web Application. Each Source is configured to collect files in a specific way. Choose the type of Source you want to create:

  • Local File Source
  • Remote File Source
  • Syslog
  • Local Windows Event Log
  • Remote Windows Event Log
  • Script Source

Alternately, you can configure a Script Action to collect your data. Learn more in Collecting from a Script Action.
