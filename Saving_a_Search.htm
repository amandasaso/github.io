Saving a Search

Whether you are running ad hoc searches during a forensic investigation or running standard searches for health checks, you can save any search query for reuse. When you save a search, you have the option to set up the saved search to run at a scheduled interval with an automated notification by email of the search results. You can edit a saved search at any time.

To save a search:

  1. In the Search tab, after typing your search query, click Save As below the search field.


  1. For Search name, enter a name for your Saved Search. If you'd like, type an optional description to help you identify this search. (Optional.)
  2. The search query populates automatically in the Search field. You can make changes to the search syntax or query details if you need to.
  3. Choose a Timerange option that will be the default range when you run the saved search.
    If you'd like to search backwards, you can type a timerange like -15m.
  4. Choose a Folder to save your search. To add a new Folder to the Content Library, click the blue "+" and name the new folder.



  5. If you'd like to set a schedule for this search, or to set the search to run periodically with an optional alert, set Schedule options.
  6. Click Save.

The search query is added to your Content Library.
