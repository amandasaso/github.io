Editing a Collector

From the Collectors tab of the Sumo Logic Web Application, you can edit some characteristics of a Collector, including its name, description, Host Name, and Category.

Changes to metadata are applied to messages going forward from this point in time, and aren't applied retroactively,

To edit the name or metadata fields for a Collector:

  1. In the Sumo Logic Web Application select Manage > Collectors.







  1. Click the Collector name, or click the Edit link to the right of the Collector name.
  2. Change the name or change the metadata fields as needed. If you set the Host Name or Category value at the Collector level, then all Sources belonging to this Collector are tagged with these metadata fields. If you later specify metadata at the Source level, the Collector metadata will be overwritten.



  3. Click Save to apply your changes.
